# Portable Turtlestitch / Snap! 12 Lasso Extension — v3.0

This document packages the complete **Lasso Multi-Selection v3** feature into all portable formats for Turtlestitch and Snap! 12.

All methods work on:
- **Turtlestitch:** `https://www.turtlestitch.org`
- **Snap! 11:** `https://snap.berkeley.edu`

---

## 1. Browser Extension (Chrome / Firefox — Manifest V3)

Create a folder `snap11-lasso-extension` and place these three files inside it.

### `manifest.json`

```json
{
  "manifest_version": 3,
  "name": "Turtlestitch / Snap! 11 Lasso Selection",
  "version": "2.0",
  "description": "Adds rectangle lasso multi-selection, Alt-click toggle, Select All (Cmd+A/Ctrl+A), and Backspace/Delete shortcuts to Turtlestitch & Snap! 11 workspaces.",
  "permissions": ["activeTab"],
  "content_scripts": [
    {
      "matches": [
        "https://snap.berkeley.edu/snap/snap.html*",
        "https://www.turtlestitch.org/*",
        "http://localhost:8000/*",
        "http://127.0.0.1:8000/*"
      ],
      "js": ["content.js"],
      "run_at": "document_end"
    }
  ],
  "web_accessible_resources": [
    {
      "resources": ["inject.js"],
      "matches": ["<all_urls>"]
    }
  ],
  "browser_specific_settings": {
    "gecko": {
      "id": "snap-lasso-selection@antigravity.addon"
    }
  }
}

```

### `content.js`

```javascript
const script = document.createElement('script');
script.src = (typeof browser !== 'undefined' ? browser : chrome).runtime.getURL('inject.js');
script.onload = function() {
    this.remove();
};
(document.head || document.documentElement).appendChild(script);

```

### `inject.js`
See full unminified source: [`extensions/lasso_selection/inject.js`](extensions/lasso_selection/inject.js)

- ✅ **NEW: Deep Stack Selection** — Bounding box logic now uses `fullBounds()`, meaning lassoing any part of a stack selects the entire script.

- 🐛 **FIX:** Implemented aggressive garbage collection on `mouseDownLeft` to permanently eradicate orphaned "ghost lasso" bounding boxes.
- 🛠️ **ARCH:** Replaced volatile `constructor` checks with prototype fingerprinting (`isLassoGroup`) for 100% live-reload resiliency.
